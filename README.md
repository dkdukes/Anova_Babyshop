## Description
Welcome to Anova Baby Shop, your trusted destination for quality baby products. We are passionate about helping parents find safe, comfortable, and affordable items for their little ones. From baby clothing and accessories to essential care products, we carefully select items that support your baby’s growth and happiness.<br/>
At Anova Baby Shop, we believe every baby deserves the best. Our goal is to provide parents with reliable products, friendly service, and a convenient shopping experience. Whether you are preparing for a newborn or shopping for your growing child, we are here to support you every step of the way.
## Setup
- Clone the code <br/>
- Use git pull to have the code on your local machine <br/>
-Run json-server --watch db.json command to have the local json db live
- Run the code using your preferred IDE
